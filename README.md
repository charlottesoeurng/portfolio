# Charlotte's Portfolio

Personal website built with Vue 3 and Vite, deployed on GitHub Pages.

## Tech Stack

- Vue 3 (Composition API)
- Vite
- CSS custom properties

## Structure

- `src/components/` — About, Skills, Projects, Resume, NavBar
- `src/assets/` — Per-component stylesheets
- `public/` — Static assets (images, resume PDF)

## Development

```bash
npm install
npm run dev
```

## Deployment

```bash
npm run build
```
